# ContextPulse — Product Naming

**Selected name:** ContextPulse
**Date:** 2026-03-15
**Status:** Name selected — domain not yet secured

---

## Why ContextPulse

- **Context** is the defining concept of AI utility — context windows, context engineering, Model Context Protocol. Every developer already uses this word when discussing AI tools. It signals exactly what the product does without explanation.
- **Pulse** conveys always-on, heartbeat, organic workflow. The tool runs continuously in the background, synthesizing what you're working on so your AI agents can help in real time.
- **Platform potential:** Screen capture is product #1 ("visual context"), but the brand scales to memory context, agent context, project context, global context.

Tagline ideas:
- "Always-on context for AI agents"
- "Your AI sees what you see"
- "The pulse of your workflow"
