ContextPulse Sight

Continuous context for MCP agents

Sight captures your desktop and serves it to Claude Code, Cursor, Gemini CLI, or any MCP-compatible agent. No manual screenshots. No broken flow.

  • Always-on daemon — auto-captures every 5 seconds in the background
  • Per-monitor capture with rolling buffer and change detection
  • Built-in OCR — extract text from any screen without sending images
  • Activity database with full-text search across window titles and OCR
  • Time-travel context — "what was on my screen 10 minutes ago?"
  • Clipboard monitoring with history and search
  • Diff-aware capture with pixel-level change scoring
  • Token cost estimation — know what each capture costs before sending
  • Multi-agent awareness — tracks which agents consume your context
  • OCR redaction — API keys, tokens, and passwords are automatically stripped
get_screenshot() Capture active, all, or specific monitor
get_screen_text() OCR current screen at full resolution
get_recent() Recent buffer frames with diff filtering
get_buffer_status() Health check + token cost estimates
get_activity_summary() App usage distribution over time
search_history() FTS5 search across titles + OCR text
get_context_at() Frame + metadata from N minutes ago
get_clipboard_history() Recent clipboard text entries
search_clipboard() Search clipboard history by content
get_agent_stats() Per-agent MCP usage statistics

FAQ

Common questions

Under 1% CPU and under 20MB RAM. The daemon uses mss for capture (3ms per frame) and only stores frames that actually changed. You'll forget it's running.
Window blocklist lets you skip specific apps (banking, password managers). Captures auto-pause on lock screen. OCR redaction strips API keys, tokens, and passwords from extracted text. Everything stays local — no cloud, no telemetry.
Any MCP-compatible tool. That includes Claude Code, Cursor, Gemini CLI, and any custom agent using the MCP SDK. The --setup flag auto-generates the right config for each.
Screenpipe records everything you see, say, and hear — full video and audio. ContextPulse captures only what your AI agents need: screen context. That's why it uses <1% CPU vs Screenpipe's 5-15%, and <20MB RAM vs 200-500MB. It's also $29 vs $400.
Windows-first, but the capture library (mss) is cross-platform. macOS and Linux support are planned for a future release.
Four modes: active monitor (Ctrl+Shift+S), all monitors (Ctrl+Shift+A), cursor region (Ctrl+Shift+Z), and auto-timer (every 5 seconds with event-driven triggers on window focus change, monitor boundary cross, and activity after idle).
